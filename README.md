# My University App

An Android application demonstrating core Android development concepts including Networking, SQLite persistence, Theme management, and WebView integration. Ideal for beginners to understand practical Android app architecture and features.

---

## Features

- **Themes**: Light, Dark, and Custom themes with persistent user preference.
- **Offline Support**: Caches API data to SQLite for offline access.
- **REST API Integration**: Fetches data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/) to demonstrate networking.
- **WebView**: Integrated internal browser for loading web content.
- **Login System**: Simple login with validation.
- **SQLite Persistence**: Store and retrieve data locally.
- **User-friendly UI**: Material Design components for better UX.
