# My University App

An Android application demonstrating core Android development concepts including Networking, SQLite persistence, and Theme management.

## Features
- **Themes**: Light, Dark, and Custom themes (Persisted).
- **Offline Support**: Caches API data to SQLite for offline access.
- **REST API**: Fetches data from JSONPlaceholder.
- **WebView**: Integrated internal browser.

## Tech Stack
-   **Language**: Java
-   **HTTP Client**: Retrofit
-   **Database**: SQLite (OpenHelper)
-   **Architecture**: MVC

## Setup
Import the project folder into Android Studio and run.
-   **Login Credentials**: `admin` / `1234`
